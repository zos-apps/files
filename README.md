## About

File manager for browsing and organizing your files.

## Features

- File browsing
- Icon/list views
- Quick Look
- Drag & drop
- Search

## Installation

### From App Store
Open the **App Store** in zOS and search for "Files".

### Via npm
```bash
npm install @zos-apps/files
```

## Usage

- Navigate folders
- Double-click to open
- Drag to move files

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Cmd+N` | New window |
| `Cmd+I` | Get info |
| `Space` | Quick Look |


## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
